FocusFlow

When you focus, flows!

This is a focus app where you focus to get coins! If you leave the app, you fail, and you lose your hard-earned coins. With the coins, you can buy many things, including (for now) break passes and app icons.

For a detailed list of features, both current and planned, check out the README!

Installation

Join the TestFlight to try the app! (Note: if the link doesn't work, it's because Apple is still approving my build.) Or, clone the repo and run the project in Xcode.
